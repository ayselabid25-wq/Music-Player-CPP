# Music-Player-CPP
This project is a console-based Music Player developed in C++ as part of my Data Structures course. The application allows users to manage a playlist by adding, deleting, searching, and playing songs while demonstrating the implementation of various data structures. 
Features:
• Add Songs
• Delete Songs
• Search Songs
• Display Playlist
• Play Songs
• Queue Management
• Stack Operations
• Playlist Navigation
Data Structures Used:
• Linked List
• Stack
• Queue
Skills Demonstrated:
• Object-Oriented Programming
• Data Structures
• Problem Solving
• File Handling
• Algorithm Design
